Learning The Traffic Rules

This Website represents the project developed to create a new way of learning the traffic rules. The idea of the project came from the Ministry of transportations Hackathon projects. The website contains an interactive simulation along with a quiz to learn basic traffic rules laid down to facilitate the traffic on the roads. The Development of this website is done by using Angular JS in combination with a Unity based simulation which provide a fun way to learn the traffic rules.

The website can be used by anyone without any prior training in the specific simulation. The simulation is provided with a fair amount of instructions along with it.
